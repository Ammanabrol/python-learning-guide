üìù Step 2: Variables and Data Types

What are Variables?

Variables are containers that store data. Think of them as labeled boxes where you can put information.

# Text (String) client_name = "John Smith" case_type = "Personal Injury" # Numbers (Integer and Float) case_number = 2024001 settlement_amount = 150000.50 # True/False (Boolean) case_active = True case_settled = False

Data Types in Python

  • String (str): Text data - "Hello", "Case ABC"
  • Integer (int): Whole numbers - 42, 2024
  • Float: Decimal numbers - 3.14, 150000.50
  • Boolean (bool): True or False

‚öôÔ∏è Step 3: Functions

What are Functions?

Functions are reusable blocks of code that perform specific tasks. They help organize your code and avoid repetition.

def greet_client(name): """Function to greet a client""" return f"Hello, {name}! Welcome to our law firm." # Using the function message = greet_client("John Smith") print(message)

Functions with Multiple Parameters

def calculate_attorney_fees(settlement, percentage): """Calculate attorney fees from settlement""" fee = settlement * (percentage / 100) client_amount = settlement - fee return fee, client_amount # Using the function attorney_fee, client_gets = calculate_attorney_fees(100000, 33.33) print(f"Attorney Fee: ${attorney_fee:.2f}") print(f"Client Gets: ${client_gets:.2f}")

üìã Step 4: Lists and Dictionaries

Lists - Storing Multiple Items

Lists store multiple items in a single variable, like a filing cabinet with multiple folders.

# List of case documents documents = ["complaint.pdf", "motion.docx", "evidence.jpg", "contract.pdf"] # Accessing list items first_document = documents[0] # complaint.pdf last_document = documents[-1] # contract.pdf # Adding to list documents.append("witness_statement.pdf") # List length total_documents = len(documents) print(f"Total documents: {total_documents}")

Dictionaries - Key-Value Pairs

Dictionaries store data in key-value pairs, like a real dictionary where you look up words (keys) to find definitions (values).

# Client information dictionary client = { "name": "John Smith", "phone": "555-0123", "email": "john@email.com", "case_type": "Personal Injury", "status": "Active" } # Accessing dictionary values client_name = client["name"] client_phone = client["phone"] # Adding new information client["attorney"] = "Jane Doe" client["filing_date"] = "2024-01-15" # Dictionary length total_fields = len(client) print(f"Client has {total_fields} fields of information")

Document Management System

class LegalDocument: def __init__(self, name, doc_type, client, date_created): self.name = name self.doc_type = doc_type self.client = client self.date_created = date_created self.status = "Draft" def finalize_document(self): self.status = "Final" print(f"{self.name} has been finalized") def get_info(self): return f"{self.name} ({self.doc_type}) - {self.client} - {self.status}" # Create documents contract = LegalDocument("Service Agreement", "Contract", "ABC Corp", "2024-01-15") motion = LegalDocument("Motion to Dismiss", "Motion", "John Smith", "2024-01-16") print(contract.get_info()) contract.finalize_document() print(contract.get_info())

Time Tracking and Billing

def calculate_legal_bill(time_entries, hourly_rate): """Calculate total legal bill from time entries""" total_time = 0 bill_details = [] for entry in time_entries: date = entry["date"] hours = entry["hours"] description = entry["description"] amount = hours * hourly_rate total_time += hours bill_details.append({ "date": date, "hours": hours, "description": description, "amount": amount }) total_amount = total_time * hourly_rate return { "details": bill_details, "total_hours": total_time, "total_amount": total_amount } # Example usage time_entries = [ {"date": "2024-01-15", "hours": 3.5, "description": "Client consultation"}, {"date": "2024-01-16", "hours": 2.0, "description": "Document review"}, {"date": "2024-01-17", "hours": 4.5, "description": "Court preparation"} ] bill = calculate_legal_bill(time_entries, 250) print(f"Total Hours: {bill['total_hours']}") print(f"Total Amount: ${bill['total_amount']:,.2f}")
